there are two files because sanger produces a forward and a reverse read

FASTQ format starts with a @ instead of a > like fasta. comments are still in (). The next line is the actual sequence followed by a new line an +. After the + is the quality score. quality score determines how confident the nt is. Quality is based of numbering translated to ascii, A = 64. Its diff for each format but overall all same idea

@title (comment)
ATGCAT 
+
AA/EEE (score)

Mapping RNA seq data to a genome

Effectively using a known genome as a guide to align each fragment and map them to different genes in the genome. Based on the number of sequence reads we can then determine expression levels.

Q10: Where are most the accepted hits located?

Q11: Following Q10, is there any interesting gene around that area?
